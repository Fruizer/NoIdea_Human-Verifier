# Noldea - Human Writing Verifier

**Noldea** is a browser-based, "Proof of Work" text editor designed to verify human authorship. 

Standard AI detectors try to guess if a text is AI-generated by analyzing the final words, which often leads to false positives. Noldea takes a different approach: it proves the text was written by a human by tracking the **behavior** of the writer in real-time. 

By analyzing Words Per Minute (WPM), natural pauses (thinking time), backspaces, and copy-paste events, Noldea generates a comprehensive Authorship Report with a dynamic Confidence Score.

## Key Features

* **Behavioral Tracking Engine:** Monitors active typing time, natural pauses (>2 seconds), character insertions, and deletions.
* **Paste Detection:** Flags large copy-paste events (15+ words) and reminds the author to provide citations.
* **Rule-Based Suggestion Scanner:** A lightweight, non-ML scanner that flags repeated words, filler words (e.g., "literally", "basically"), and run-on sentences.
* **Dynamic Checkpoints:** Automatically reads `H1`, `H2`, and `H3` tags to generate a clickable, smooth-scrolling Table of Contents on the side panel.
* **Draft History & Auto-Save:** Automatically snapshots the document every 25 seconds. Users can restore previous versions or delete old drafts.
* **Rich Text Formatting:** A built-in Google Docs-style toolbar for fonts, sizes, headers, and alignment.
* **Export Options:** Download the final written paper as a clean PDF (with UI elements hidden) and export the raw behavioral telemetry as a `.txt` report.
* **Seamless Dark/Light Mode:** Full theme toggling using CSS variables.

## Tech Stack

This project was built as a lightweight, frictionless web app with zero dependencies. 

* **HTML5:** Structure and `contenteditable` workspace.
* **CSS3:** Styling, flexbox layouts, `@media print` queries for PDF generation, and CSS variables for theming.
* **Vanilla JavaScript (ES6+):** The core tracking engine, event listeners, DOM manipulation, and math logic. 

## How to Use It

Because Noldea is built with pure web technologies, there are no complicated installations, plugins, or local servers required.

1. Clone this repository or download the ZIP file.
2. Open the folder on your computer.
3. Double-click `index.html` to open it in any modern web browser (Chrome, Edge, Firefox, etc.).
4. Start typing! When you are finished, click **Generate Report** to view your Confidence Score.

## The "Proof of Work" Scoring Logic

The final Confidence Score starts at 100% and deducts points based on unnatural typing behaviors:
* **Pauses:** Too many long pauses (thinking time) are natural, but zero pauses indicate a bot. 
* **Large Pastes:** Heavy penalties for dropping in large chunks of pre-written text.
* **Speed:** Deductions for sustaining highly unnatural WPM speeds. 
* **Grammar/Flow:** Minor deductions for leaving in repetitive text or excessive filler words flagged by the scanner.
